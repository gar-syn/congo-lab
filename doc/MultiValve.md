## Serial Commands
```
Description - Get Current Position
Command - IN_SP_00\r\n
Unit - N/A
Reply - IN_SP_00 +00\r\n
Value - N/A
```
```
Description - Set Current Position
Command - OUT_SP_00 00\r\n
Unit - N/A
Reply - OUT_SP_00 +00\r\n
Value - N/A
```
